# NEO Tic-Tac-Toe
Yet another Tic-Tac-Toe application for cordova.

## Description
NEO Tic-Tac-Toe is simple but more exciting Tic-Tac-Toe on [OnsenUI](https://onsenui.io).

One additional rule: only 6 elements are kept on the board.
